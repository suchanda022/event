# Event Services Website

A React-based website showcasing event services including production setup, media coverage, stage effects, government infrastructure, and electrical services.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm

### Installation

1. Clone the repository
2. Move into the frontend app folder: `cd frontend`
3. Run `npm install` to install dependencies

### Development

Run `npm run dev` from the `frontend` folder to start the development server.

### Build

Run `npm run build` from the `frontend` folder to build the project for production.

## Project Structure

- `frontend/src/App.jsx`: Main application component with service sections
- `frontend/src/App.css`: Styles for the website
- `frontend/public/`: Static assets
- `frontend/dist/`: Build output (after running build)

## Features

- Responsive design
- Sections for different event services
- Placeholder areas for images (to be replaced with actual images)

## Customization

Replace the placeholder divs in `frontend/src/App.jsx` with actual images for each service section.
